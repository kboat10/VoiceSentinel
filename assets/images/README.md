# Images

Put your **Voice Sentinel company logo** here.

**Filename to use:** `voice_sentinel_logo.png`

- Supported formats: PNG, SVG, or JPG (PNG recommended for transparency).
- You can copy the same file from your Flutter app:  
  `Voice Recording App GUI/assets/images/voice_sentinel_logo.png` → paste it here.
- If you use a different filename, update the image path in `index.html` (search for `voice_sentinel_logo`).

The logo is used on:
- Welcome / Sign in & Sign up pages
- Sidebar (when logged in)
